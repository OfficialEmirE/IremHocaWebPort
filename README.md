## 🛠️ Kurulum ve Geliştirme

Projeyi kendi bilgisayarınızda açmak ve düzenlemek için aşağıdaki adımları izleyin:

1. **Unity Sürümü:** Bilgisayarınızda [Unity Hub](https://unity.com) üzerinden **Unity 2021.3 LTS** sürümünün yüklü olduğundan emin olun.
2. **Projeyi İndirin:** Bu depoyu (repository) ZIP olarak indirin veya git ile clone'layın.
3. **Unity'ye Ekleyin:** Unity Hub'ı açın, "Add" butonuna basın ve indirdiğiniz klasörü seçin.

> [!TIP]  
> Konsolda (Log) `UnityEngine.UI.dll` veya `TextMeshPro` (TMP) kaynaklı çok sayıda hata görüyorsanız, bu genellikle kütüphane referanslarının kopmasından kaynaklanır. 
> 
> **Çözüm:** İlgili `.dll` dosyasını seçip Inspector üzerinden **"Auto Reference"** seçeneğinin kapalı olduğundan emin olun veya hata veren paketleri Package Manager üzerinden "Re-import" edin.
